# Chronovita 研发路线图

> 当前版本：**v0.1.0（平台壳重启）**。前序所有版本因转向"框架先行"而归档清空。

## 阶段 0 · 平台壳层（v0.1.x）
**目标：搭好五模块平台框架，没有任何具体课程内容。**

- [v0.1.0] 五模块平台壳落地
  - 顶栏（Logo + 5 模块导航 + 搜索 + 用户信息空位）
  - 路由：`/` · `/courses` · `/learning` · `/practice` · `/profile`
  - 5 个占位页骨架，深海蓝 + 金色主题
  - 后端：5 个占位路由 + 健康检查 + 持久化基础
  - Pencil 仿绘 5 模块设计稿
  - 文档基线重写

## 阶段 1 · 课程数据模型（v0.2.x）
- 课程 / 单元 / 章节 / 节四级数据模型
- 课程中心列表与详情接口
- "我的学习"进度模型与持久化
- 课程目录侧栏交互（仿 UI 截图）

## 阶段 2 · 看 · 沉浸叙事（v0.3.x）
- 视频播放器壳 + AI 生成入口
- 关键词词汇卡组件
- 章节切换与进度追踪

## 阶段 3 · 练 · 实践课堂（v0.4.x）
- 决策推演通用引擎（不绑定具体剧本）
- 实践课堂 UI（依据 `UI/实践课堂.png`）
- 课堂内即时反馈

## 阶段 4 · 问 · 双模智者（v0.5.x）
- LLM 适配层（OpenAI / DeepSeek / Ollama / mock）
- 同伴 + 专家双 Persona 框架
- 引证后置由后端控制

## 阶段 5 · 创 · 知识谱系（v0.6.x）
- 知识图谱画布通用框架
- 课程级节点种子从课程数据生成

## 阶段 6 · 课程内容填充（v0.7.x ~）
- 由用户后续指令逐节推进
- 每节课打通"看练问创"四层闭环
